Como realizar una clase en Java

Para crear una clase en java utilizando como IDE NetBeans debemos de seguir los siguientes pasos:
  • Primero debemos de crear un nuevo proyecto en java.



  • Segundo crear una clase.



    Se muestra la sintaxis de una clase en Java.



  • Tercero definir los atributos de la clase aunque estos puede o no tenerlos.


  • Cuarto crear los contructores.

  • Quinto crear los metodos.

    Entre las lineas 47 y 50 aparece un método . El método da acceso publico (public), se llama escribirInformacion y no tiene parametros. El método tiene una unica sentencia que ocupa las líneas 48 y 49 y va a mostrar en consola un texto.
